Hubspot Off-Canvas Form

A lightweight plugin for off-canvas forms

Demo

Off-Canvas is lightweight, easy to implement, and compatible with IE7+, Chrome, Firefox, and Safari

Template

  1. Navigate to the landing page you wish to implement Off-Canvas with and click Edit.
  2. Click the Settings Tab and click on the Advanced Options dropdown.
  3. Click Edit Template.
  4. Create a new Module Group and place it at the very bottom of the page. Create a Custom HTML field and a Form Field and place them inside the Module Group you just created.

    5. Note: If you already have a form within your template, delete it and follow the instructions above.

  5. Publish your template changes.

Custom HTML Close Button

  1. Navigate back to your landing page and click the Edit Modules icon in the sidebar menu.
  2. Select the last custom HTML module in your list of modules.
  3. Copy and paste the following code into the custom HTML textarea and save your project.
    4. <div class="modal-close-bar">
  <div class="button-wrapper">
    <span class="close-button"></span>
  </div>
</div>

Form Styles

An external stylesheet will need to be added to your landing page in order for the plugin to function properly.

  1. Navigate to the landing page you are implementing Off-Canvas with and click Edit.
  2. Click the Settings Tab and click the Advanced Options dropdown.
  3. Scroll down to Stylesheets and locate Page Stylesheets.
  4. Click Attach a stylesheet
  5. From the dropdown list, select Custom/pages/css/off-canvas-form-styles.css.

    6. If you have more than one CSS file attached, make sure the Off-Canvas stylesheet is the last in the list.

Form Scripts

An external JavaScript file will need to be added to your landing page in order for the plugin to function properly.

  1. Navigate to the landing page you are implementing Off-Canvas with and click Edit.
  2. Click the Settings Tab and click the Advanced Options dropdown.
  3. Scroll down to Footer HTML.
  4. Copy and paste the following tokens into the Footer HTML textarea and save your landing page.
    5. <script src='https://cdnjs.cloudflare.com/ajax/libs/jquery/3.2.1/jquery.min.js'></script>
<script src='{{ get_public_template_url('Custom/page/js/off-canvas-form-scripts.js') }}'></script>

    If you have other script files in your footer, make sure the tokens above are at the top of the list.

Initialize Plugin

Simply add the class name "schedule-a-demo-cta" to any button to link it to your Off-Canvas form.
This may need to be done with JavaScript; if so, contact Alex for assistance.
